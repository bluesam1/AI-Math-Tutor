# Story 3.12: Logo-Based Color System Foundation

## Status

Pending

## Story

**As a** student,
**I want** the application to use vibrant, cohesive colors that match the Learn Math logo,
**so that** the interface feels engaging, modern, and reflects the brand's personality.

## Acceptance Criteria

1: Color system is updated in `globals.css` with logo-based color palette extracted from Learn Math logo.

2: Primary colors are defined using HSL values:
- Medium to Dark Blue (`217 65% 35%` / `#1E3A8A`) - Primary brand color
- Lighter Blue (`217 91% 65%` / `#60A5FA`) - Secondary buttons, accents, hover states
- Bright Cyan/Sky Blue (`192 100% 50%` / `#00C9FF`) - Backgrounds, gradients

3: Accent colors are defined:
- Vibrant Lime Green (`75 80% 45%` / `#84CC16`) - Success states, celebrations, calculator accents
- Lighter Lime Green (`75 85% 55%` / `#A3E635`) - Light accent variant

4: Background colors are updated:
- Sky blue tint (`192 100% 98%`) - Primary background
- Soft grey (`220 14% 96%`) - Secondary background

5: Text colors maintain accessibility:
- Dark grey/black (`217 33% 17%`) - Primary text
- Medium grey (`217 20% 45%`) - Secondary text

6: All CSS custom properties use HSL format for consistency.

7: Color system maintains backward compatibility with existing component classes.

8: All color combinations meet WCAG AA contrast requirements for accessibility.

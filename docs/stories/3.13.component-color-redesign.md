# Story 3.13: Component Color Redesign

## Status

Pending

## Story

**As a** student,
**I want** all UI components to use the vibrant logo-based colors consistently,
**so that** the entire application feels cohesive, modern, and engaging.

## Acceptance Criteria

1: Layout component uses logo-based colors:

- Subtle gradient background (sky blue to white)
- Border colors use primary blue
- Logo is displayed in header/navigation area

2: Problem Panel uses logo-based colors:

- Background gradient uses logo-inspired colors
- Problem display card has colorful border (primary blue)
- "Change Problem" button uses primary blue with hover states
- Subtle shadows use brand colors

3: Chat Panel uses logo-based colors:

- Background uses subtle gradient
- Header uses brand colors
- Message bubbles are redesigned:
  - Student messages: Light blue background with blue border
  - Tutor messages: White/light grey with green accent border
- Input field uses primary blue focus states
- Send button uses lime green accent

4: Problem Input component uses logo-based colors:

- Submit button uses primary blue with hover effects
- Upload button uses lime green accent
- Input field borders use primary blue
- Drag-and-drop visual feedback uses brand colors

5: Message Item component uses logo-based colors:

- Student messages use logo blue (not indigo)
- Tutor messages use green accent border
- Answer badges use lime green
- Subtle shadows and rounded corners

6: Answer Input component uses logo-based colors:

- Primary action button uses lime green (matches logo's calculator green)
- Input field uses primary blue focus states
- Success states use lime green with celebration colors
- Error states use soft red (age-appropriate)

7: Empty State component uses logo-based colors:

- Gradient background (sky blue to white)
- Text colors use brand colors
- Call-to-action elements use brand colors

8: Visual feedback components use logo-based colors:

- Celebration Message: Lime green background
- Encouragement Message: Light blue background with lime green accents
- Error Message: Soft red with primary blue retry buttons
- Progress Indicator: Gradient from primary blue to lime green

9: All buttons consistently use brand colors:

- Primary buttons: Medium to dark blue
- Secondary buttons: Lime green
- Text/outline buttons: Primary blue border

10: All form elements use brand colors:

- Input borders: Light grey default, primary blue focus
- Checkboxes/radio: Primary blue for checked state
- Loading states: Use brand colors
